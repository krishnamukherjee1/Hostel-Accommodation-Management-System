# Hostel Accommodation Management System

A modular **Hostel Accommodation Management System** developed in **C++** using **Object-Oriented Programming (OOP)** principles and modern software engineering practices.

This project is being built incrementally to simulate the workflow of a real software development project, with proper version control, clean architecture, and modular design.

---

## Features

### Student Management

- Add Student
- Display All Students
- Search Student by ID
- Delete Student
- Duplicate Student ID Validation
